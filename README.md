# [FreqEdit: Preserving High-Frequency Features for Robust Multi-Turn Image Editing]

[![Project Page](https://img.shields.io/badge/Project-Page-green.svg)](https://freqedit.github.io/)
[![arXiv](https://img.shields.io/badge/arXiv-2512.01755-b31b1b.svg)](https://arxiv.org/abs/2512.01755)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)



**FreqEdit** enables consistent multi-turn image editing. Base models (FLUX.1 Kontext and Qwen-Image) exhibit progressive quality deterioration during iterative editing, including body deformations, edge over-sharpening, and texture collapse. FreqEdit addresses these limitations through strategic high-frequency reinforcement.
